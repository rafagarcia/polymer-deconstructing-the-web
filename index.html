WHAT IS POLYMER AND WHAT IS NOT (OR WASN´T)

  • Polymer IS A LIBRARY, built on top of Web Components, designed to help us creating Web Components, NOT A FRAMEWORK
  • or IT WASN´T, because with the new Polymer App Toolbox Polymer allows us to build Progressive Web Apps following the PRPL pattern

THE PRPL Pattern

  • Push critical resources for the initial route
  • Render initial route
  • Pre-cache remaining routes
  • Lazy-load and create remaining routes on demand

WEB COMPONENTS AS A STANDARD

  • Reusable user interface widgets that are created using open Web technology
  • They are part of the browser (#usetheplatform)
  • An existing Web Component can be used without writing code, simply by adding an import statement to an HTML page
  • Composed by four specifications: Custom Elements, HTML Imports, Templates, Shadow DOM

Custom Elements

This specification describes the method for enabling the author to define and use new types of DOM elements in a document

HTML Imports

HTML Imports are a way to include and reuse HTML documents in other HTML documents

Templates

This specification describes a method for declaring inert DOM subtrees in HTML and manipulating them to instantiate document fragments with identical contents

Shadow DOM

This specification describes a method of establishing and maintaining functional boundaries between DOM trees and how these trees interact with each other within a document, thus enabling better functional encapsulation within the DOM

“We are not longer designing pages; we are designing systems of components” @stephenhay

ATOMIC DESIGN. Definition

Atomic design is a methodology created by Brad Frost to help us building design systems, structured in five levels:

  1. Atoms
  2. Molecules
  3. Organisms
  4. Templates
  5. Pages

Atoms

Atoms are the basic building blocks of matter. Applied to web interfaces, atoms are our HTML tags, such as a form label, an input or a button.

Molecules

Molecules are groups of atoms bonded together and are the smallest fundamental units of a compound. For example, a form label, input or button combined together as a form

Organisms

Molecules give us some building blocks to work with, and we can now combine them together to form organisms. Organisms are groups of molecules joined together to form a relatively complex, distinct section of an interface

Templates

Templates consist mostly of groups of organisms stitched together to form pages

Pages

Pages are specific instances of templates

Polymer Library

  1. Using the Polymer Catalog
  2. Creating an element from scratch

1. Using the Polymer Catalog

The Polymer Element Catalog provide elements that you can use in your web pages and apps. These elements are built with the Polymer library.

  • App Elements => App elements
  • Iron Elements => Polymer core elements
  • Paper Elements => Material design elements
  • Google Web components => Components for Google´s APIs and services
  • Gold Elements => Ecommerce elements
  • Neon Elements => Animation and special effects
  • Platinum Elements => Offline, push and more
  • Molecules => Wrappers for third party libraries
